RoomSync — Intelligent Roommate Allocation

The Challenge

Manual roommate matching at scale is bottlenecked by subjective student data and high processing times. Institutions lacked a repeatable method to process preference matrix points safely while preserving system reliability.

The Architecture

Deployed a split-responsibility 3-service backend: A high-speed Next.js dashboard, a core Node.js/Express API Gateway managing user states, and an isolated Python FastAPI engine running Scikit-learn K-Means clustering algorithm over 30+ compatibility parameters per student.

Optimization: Auto-groups compatible pairs reducing matching conflicts.
Admin Controls: Overrides, room-locks, and change-request pipelines.
Data Flow: Integrated CSV import pipeline with active Google Sheets sync.

Codemia — Automated Code Grading Engine

The Challenge

Executing untrusted user code in multiple programming languages (C++, Java, Python, JavaScript) synchronously blocks standard API gateways and creates massive security/concurrency vulnerabilities.

The Architecture

Designed a scalable, asynchronous message passing system. Submissions are pushed to Redis-backed BullMQ job queues to prevent CPU thread exhaustion on the web server. Worker instances consume queue items and route execution jobs safely to isolated sandboxed execution endpoints (Piston API), caching the output.

Queued Execution: Isolated worker processes for high-volume concurrent runs.
Secure Auth: Role-based access control with robust JWT verification structures.
Nginx Routing: Reverse proxy setup for SSL termination and static caching.

SymbiConnect — Real-Time College Chat

The Challenge

Sustaining stateful websocket connections with database-backed historical lookups without introducing I/O blockers on single-threaded Express handlers.

The Architecture

Developed an event-driven messaging gateway with Socket.io. Implemented persistent MongoDB Atlas logging to capture chat states, utilizing JWT payloads to authenticate handshakes. Optimized websocket performance boundaries by decoupling logging from broadcast execution loops.

Real-time Broadcasts: Socket event loops for low-latency messaging.
Secure Handshakes: Connection validation utilizing custom JWT validation.
Availability: Structured deployment pipelines targeting 99% uptime.
