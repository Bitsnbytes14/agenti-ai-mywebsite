# Mohammad Ahmad — Personal Portfolio (AgentiX AI Class Project)

This repository contains a single-page developer portfolio website built as part of the **AgentiX AI class** to explore, practice, and evaluate AI-assisted web development workflows.

---

## 📋 Overview

This portfolio website showcases the software engineering background, project details, and technical focus areas of Mohammad Ahmad. It was built from scratch without external frameworks or dependencies to study modern vanilla web standards in a classroom learning setting.

---

## 🎯 Purpose

This project is created and maintained solely for **learning, experimentation, and coursework validation**. It is a static, one-time project designed for educational review and is not intended to be a reusable template, a maintained open-source codebase, or a production-ready template.

---

## 🛠️ Technologies Used

- **HTML5:** Semantic structure, meta-tag SEO configuration, and ARIA accessibility features.
- **CSS3:** Responsive flexbox/grid layout structures, transition classes, and custom light/dark color tokens.
- **JavaScript:** Native ES6+ DOM manipulation for scrollspy, typing animations, canvas particle loops, and form validation.

---

## 💻 How to Run

Because this website uses native browser standards, it requires no compilers or active runtime dependencies:
1. Clone or download this repository.
2. Open the [index.html](file:///index.html) file directly in any modern web browser.
